# WagSave Samples

Sample Unity projects demonstrating WagSave integration.

## Samples

### [Simple](Samples/Simple/)

A minimal 2D arena game. Move a player around, collect enemies to score points, and use Save/Load to persist game state. Covers the core WagSave API: `Save()`, `Load()`, and the `OnSaveCompleted` / `OnLoadCompleted` callbacks.

| Requirement | Version |
|-------------|---------|
| Unity | 6000.0.71f1 (Unity 6) |
| Render Pipeline | Universal Render Pipeline (URP) 17.0.4 |
| Input System | 1.19.0 |
| WagSave | Latest |

See [Samples/Simple/README.md](Samples/Simple/README.md) for setup and usage instructions.
